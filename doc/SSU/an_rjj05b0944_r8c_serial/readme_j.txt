このREAD MEファイルでは、本フォルダの構成、内容について説明しています。

 an_rxxxxxxxxxx_YYYY_ZZZZ	全てを格納するフォルダ
   ├ rxxxxxxxxxx_AAAA.pdf	アプリケーションノート本体ファイル
   ├ readme_j.txt 		本READ MEファイル
   ├ workspace 		ワークスペース用フォルダ
   └ source			ソースファイル、incファイル等格納フォルダ

 YYYY：m16c、r8c等
 ZZZZ：機能名
 AAAA：ファミリ名、シリーズ名、グループ名等

ワークスペース用フォルダには、sourceフォルダのファイルを含む、統合開発環境High-performance
Embedded Workshopで動作させるために必要な全てのファイルが格納されています。

ワークスペースをご使用いただくことにより、統合開発環境High-performance Embedded
Workshopで新規ワークスペースを作成することなく、すぐにビルド、デバッグすることが可能です。

sourceフォルダのファイルは、新規でワークスペースを作成される場合、もしくは、弊社パートナー
メーカ製開発環境でご使用いただけます。なお、パートナーメーカ製開発環境では、記述方法が
異なる場合がありますので、ご使用になるツールのマニュアル等でご確認ください。

ご使用上の注意
　ワークスペースは、ご使用の統合開発環境(High-performance Embedded Workshop、
　Cコンパイラ、エミュレータソフトウエア)のバージョンによって使用できない場合があります。
　その場合、最新の統合開発環境にバージョンアップしていただくか、バージョンはそのままで
　新規ワークスペースを作成後、sourceフォルダのファイルをご使用ください。
